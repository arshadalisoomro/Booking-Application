Booking-Application  
===================  
  
A simple booking for specific flight application.  
  
1. Files structure  
/ root  
 │─ Passenger.java  
 │─ Booking.java  
 │─ SpecificFlight.java  
 └─ Main.java  
  
2. Detail for files structure  
● 本系統並沒有用到資料庫來儲存資料，亦沒有用I/O的方式來存取資料。本系統在執行時，把使用者輸入的旅客資料存放在記憶體。  
● Main 是主要執行的程式，並且負責處理GUI與使用者互動。
